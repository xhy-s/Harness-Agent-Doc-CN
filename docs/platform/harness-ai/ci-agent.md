# CI Agent

Harness AI 可以分析日志文件并将错误消息与已知问题相关联。此功能使开发者能够快速故障排除和解决部署失败，节省了筛选数百万行日志的时间。Harness AI 还会在构建开始之前建议修复并预测代码中的潜在错误。此功能旨在跨 Harness 的 CI 和 CD 产品工作。

## 开始使用 CI 的 Harness AI

在您的 Harness 账户中启用 Harness AI for CI：

1. 进入 **Account Settings**。
2. 选择 **Account Resources**。
3. 选择 **Harness AI** 磁贴。
4. 启用 **Harness AI** 设置。

启用 Harness AI 后，当步骤失败或当空 stage（没有步骤的 stage）遇到失败时，AI 驱动的错误分析和修复变得可用。

查看失败步骤或 stage 的日志时，选择 **Harness AI** 对话框以查看错误分析和故障排除建议。

步骤日志可在 **Build details** 页面上获取。
